# MyPokedex
Uma aplicação móvel estática desenvolvida com React Native e Expo, que simula uma Pokédex moderna.
Exibe informações detalhadas sobre Pokémon, como:

- Nome e número
- Imagem ilustrativa
- Tipos com cores personalizadas
- Altura, peso e categoria
- Linha evolutiva com imagens
- Descrição geral do Pokémon

Este projeto foi desenvolvido para **fins acadêmicos**, com o propósito de praticar:

- Estruturação de telas em React Native
- Estilização com base em layouts visuais
- Organização de componentes e arquivos

## Tecnologias utilizadas

- React Native
- Expo
- JavaScript
